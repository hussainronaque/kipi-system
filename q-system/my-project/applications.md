# Applications Tracker

> Every job application, outreach, and recruiter conversation in one place.
> Updated manually or via /q-debrief when pasting interview notes/recruiter convos.

## Status Legend

| Status | Meaning |
|--------|---------|
| `outreach` | DM/email sent to recruiter or HM — no application yet |
| `applied` | Application submitted |
| `screening` | Recruiter call scheduled or completed |
| `interview` | Technical or hiring manager interview stage |
| `offer` | Offer received |
| `rejected` | Rejected or ghosted after 3+ weeks |
| `withdrawn` | You withdrew |

## Rules

- Every application gets a row immediately after submitting
- Every recruiter DM gets a row in Outreach Queue before sending
- Move to Closed within 48h of final outcome
- If no response in 14 days → mark `rejected`, move to Closed
- Friday: update Weekly Stats before EOD
